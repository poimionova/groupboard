# GroupBoard v0.1.0 — Release Notes

## 🚀 Первый рабочий прототип

**Дата релиза:** май 2026  
**Тип:** Initial Release  
**Ветка:** `main`

---

## ✅ Что реализовано в этой версии

### Авторизация
- Регистрация и вход по логину/паролю
- JWT-токены (срок действия 7 дней)
- Роли: `admin`, `head` (Староста), `member` (студент)
- Защищённые маршруты — без токена доступа нет

### Группы
- Создание группы с автогенерацией 8-значного кода приглашения
- Вступление в группу по коду
- Просмотр участников с рейтингом активности

### Домашние задания
- Добавление ДЗ с дедлайном, предметом и описанием
- Отметка выполнения с прогресс-баром по группе
- Геймификация: +10 баллов за выполнение
- Фильтрация: все / не выполнено / просрочено

### Расписание
- Добавление пар по дням недели
- Выделение текущего дня
- Управление доступно только Старосте/admin

### Очередь выступлений
- Создание очереди с указанием слота (5/10/15/20/30 мин)
- **Авто-распределение** 30 студентов по датам (формула: ⌊90 ÷ слот⌋ чел/день)
- Ручная запись на свободную дату
- Swap-запросы — обмен датами между студентами
- +5 баллов за запись в очередь

### Канбан-доска
- Три колонки: К выполнению / В процессе / Готово
- Drag & drop карточек между колонками
- Приоритеты (низкий / средний / высокий)
- Шаблоны задач для старосты
- +15 баллов за завершение задачи

### Голосования
- Создание опроса с произвольными вариантами
- Анонимный и открытый режим
- Автозакрытие по дате
- +2 балла за участие

### Дашборд
- Карточки: участники, % выполнения ДЗ, задачи по статусам
- Топ-5 студентов по баллам активности
- Ближайшие дедлайны (следующие 7 дней)

## 📦 Установка и запуск

```bash
git clone https://github.com/poimionova/groupboard.git
cd groupboard
docker compose up --build
```

Открыть: http://localhost:5173
